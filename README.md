# todo-app
- MEAN stack app for PLAR

### Current Version: 3.4.4

## Getting Started
1. ```$ npm install```
2. ```$ npm start```
3. Open browser to http://localhost:3000

## Default Usage
https://lit-coast-5603.herokuapp.com/#/login

## Alternative Usage
```$ grunt serve-dev```
