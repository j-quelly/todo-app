# todo-app
- MEAN stack app for PLAR

### Current Version: 0.0.5

## Getting Started
1. ```$ bower install```
2. ```$ npm install```
3. ```$ npm start```

## Default Usage
```$ npm start```

## Alternative Usage
```$ grunt serve-dev```
