# todo-app
- MEAN stack app for PLAR

### Current Version: 0.0.8

## Getting Started
1. ```$ npm install```
2. ```$ npm start```
3. Open browser to http://localhost:3000

## Default Usage
```$ npm start```

## Alternative Usage
```$ grunt serve-dev```
