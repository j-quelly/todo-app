# todo-app
- MEAN stack app

### Current Version: 0.0.2

## Getting Started
1. ```$ bower install```
2. ```$ npm install```
3. ```$ grunt serve-dev```

## Default Usage
```$ npm start```

## Alternative Usage
```$ grunt serve-dev```
